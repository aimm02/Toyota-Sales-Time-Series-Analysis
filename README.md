# Toyota-Sales-Time-Series-Analysis

Mini project Data Analyst: Analisis tren jangka panjang (2011-2023) penjualan mobil Toyota menggunakan Python. Mengidentifikasi tahun puncak (2013) dan anomali terendah (2020) sebagai dasar perencanaan strategis.
Sumber data untuk proyek ini berasal dari Kaggle. Dataset ini mencakup data penjualan unit mobil Toyota secara bulanan selama rentang waktu lebih dari satu dekade, yaitu dari Januari 2011 hingga tahun 2023.

Link Dataset : https://www.kaggle.com/datasets/rickyaryansah/penjualan-mobil-toyota
